Dictionary

What word would you like the definition to?
 

I wanted to create a dictionary based website, so I found this API from https://www.wordsapi.com and created this. What this website is is it allows a user to search up the definition of a word. This is done since I only included the definition option; I could have gotten more information such as synonyms, rhymes, antonyms, etc. This could be useful as a dictionary app, or it could be a useful add on for the likes of an app, like words with friends or a kindle app. This is because the app allows for a user to quickly look up a definition and a tool like this could provide quick feedback.
